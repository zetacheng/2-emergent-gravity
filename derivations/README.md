# Derivation Notes

A derivation note must be committed before its production implementation. Every
note must contain:

- the scientific question;
- assumptions;
- locked conventions;
- the analytic derivation;
- expected limiting cases;
- dimensional analysis;
- known failure modes;
- pre-registered verdicts;
- the exact scripts that implement it.

Derivations must identify whether they concern the original model, a model
extension, a phenomenological EFT, or a numerical proxy.
